# matrix-calculator

> 基本的矩阵计算器，即用矩阵名输入需要计算的表达式，程序输出结果。集成开发环境为vs2017。
- 可从文件或控制台读入矩阵数据
- 利用栈结构将中缀表达式转为后缀表达式，同时去除()
- 利用Gauss-Jordan消去法实现矩阵求逆
- 利用栈结构对后缀表达式进行单目运算|(表示求逆)和双目运算+ - *
- 对错误输入采取简单的语法分析，给出错误提示
- 定义矩阵类对各种运算做抽象封装
